# 📄 Resume Screener

An **AI-powered resume screening and ATS optimization tool** designed to help job seekers and recruiters maximize resume visibility and match alignment with job descriptions.

## ✨ Features

- **Intelligent Resume Parsing**: Extracts text from PDF and DOCX files with support for multiple formats
- **Skills Detection**: Identifies and matches 200+ technical and soft skills from an extensive skills database
- **ATS Optimization**: Provides actionable recommendations to improve Applicant Tracking System compatibility
- **AI-Powered Analysis**: Leverages HuggingFace LLM integration for intelligent resume analysis
- **Job Matching Score**: Calculates compatibility scores between resumes and job descriptions
- **Web-Based Interface**: Clean, user-friendly Flask application with responsive HTML templates
- **Rate Limiting**: Built-in security features to prevent API abuse
- **Database Support**: SQLite database for storing analysis results and user data
